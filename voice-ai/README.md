An AI that allows users to talk with their voice-agent the way they want it to be. They can choose personalities tailored to their specific types. A simple workflow of:-
Voice Input --> STT --> Gemini --> TTS --> Response

IMPORTANT:
For full functionality, API keys are required. See demo video for complete experience.
The demo video showcases a custom ElevenLabs voice. For local reproducibility, users can replace the VoiceID with any public ElevenLabs voice available to their API key.
